## openapi-to
Generate SDKs,OpenAPI to:
+ [ ] js request(jsDoc)
+ [x] ts request
+ [ ] ts interface
+ [ ] request object
+ [ ] zod
+ [x] zod decorator
+ [ ] vue-Query
+ [ ] Faker.js

OpenAPI Specifications are supported:
- swagger 2.0
- openapi 3.0
